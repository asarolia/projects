There are 3 ways in which custom objects can be created in JS:

    Using Object Initializers
    Using Constructor function
    Using Object.create method


Using object initializers

In this properties and methods are directly assigned to variable in comma seperated name value pairs

var animalObject = { type : "Invertebrates", displayType : function(){ console.log(this.type); } }

Using Constructor function

In this define the object type using constructor and then use new to create object out of it

function Animal (type){ this.type = type; } var dog = new Animal("Dog");

Using Object.create method

This creates object by selecting the prototype object from which new object to be created

var animalObject = { type : "Invertebrates", displayType : function(){ console.log(this.type); } } var animal1 = Object.create(animalObject); animal1.displayType(); var fish = Object.create(animalObject); fish.type = "Fish"; fish.displayType();
